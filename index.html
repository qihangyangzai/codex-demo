06 / Core Fixes
核心改动（P0）
三个最关键的问题，不修就跑不起来。
P0

mbridge 映射兼容

MTP 禁用后，Megatron 用 decoder.final_layernorm（单数），但映射表只有 decoder.final_layernorms.0（复数+索引）。Codex 加了一行判断，自动替换。

P0

FP16 精度配置

添加 bf16: false 和 fp16: true，初始化 grad_scale_func，注释掉 precision_aware_optimizer。

P0

Token 长度余量

余量从 2048 改为 16384，避免 prompt+response 超限。

07 / All Fixes
全部改动
20 次迭代，15 个文件。

final_layernorms.0 映射报错MTP 禁用时自动替换为 final_layernorm

Prompt length exceeds余量 2048 → 16384
